# E-Commerce
E-Commerce App named Buy-Buy with Kotlin

## Project Features
 - MVVM
 - Data Binding
 - Coroutines
 - Navigation Components
 - Hilt for Dependency Injection
 - Retrofit
 - Notifications
 - Firebase Auth & Firestore
 - Glide & Picasso
 - TabLayout & ViewPager2
